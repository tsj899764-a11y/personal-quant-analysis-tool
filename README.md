# 个人量化分析工具

## 一、项目简介

个人量化分析工具是一个基于 `Python + Streamlit` 开发的本地网页项目，用于个人场景下的 A 股股票信号分析。

本项目专注于以下分析流程：

- 市场数据获取
- 技术指标计算
- 条件规则判断
- 信号结果生成

本项目不是自动交易系统，不接券商接口，不提供下单功能，不做实盘执行，也不涉及账户连接与策略托管。

项目定位是一个本地运行的个人辅助分析工具。用户手动输入 A 股股票代码后，系统自动获取最近 250 个交易日历史行情数据，计算常见技术指标，根据预设规则进行条件判断，并输出当前信号结果与图表展示页面。

## 二、功能说明

当前版本支持以下核心功能。

### 1. 股票代码输入

用户可以手动输入 6 位 A 股股票代码，例如：

- `600853`
- `000001`
- `002594`

### 2. 历史行情获取

点击“开始分析”后，系统会自动获取该股票最近 250 个交易日左右的历史行情数据。

当前主要使用的数据字段包括：

- `date`
- `open`
- `high`
- `low`
- `close`
- `volume`

### 3. 技术指标计算

系统会基于历史行情自动计算以下常用技术指标：

- `MA5`
- `MA10`
- `MA20`
- `MA60`
- `MACD`
- `RSI`
- `5 日均量`

### 4. 条件判断

系统会对最新交易数据进行以下条件判断：

- MA5 是否上穿 MA10
- MA5 是否下穿 MA10
- MA5 是否大于 MA10
- MA10 是否大于 MA20
- 当前收盘价是否站上 MA20
- MACD 是否金叉
- MACD 是否死叉
- 当前成交量是否大于 5 日均量的 1.5 倍
- RSI 是否低于 30
- RSI 是否高于 70

### 5. 信号输出

系统会根据规则判断结果输出以下四种信号之一：

- 买入信号
- 卖出信号
- 观察信号
- 无信号

### 6. 页面展示内容

页面会显示以下信息：

- 股票名称
- 股票代码
- 当前价格
- 当前信号
- 信号原因说明
- 满足条件列表
- 不满足条件列表
- 收盘价走势图
- 均线图
- 成交量图
- MACD 图

### 7. 页面风格

页面采用深色科技金融分析风格，特点包括：

- 深蓝、深灰、黑色背景
- 青蓝色、荧光蓝、浅绿色强调色
- 卡片式布局
- 信号区域突出显示
- 图表区域风格统一
- 整体结构规整、清晰、专业

## 三、项目目录结构说明

项目目录结构如下：

```text
.
├── app.py
├── data_fetcher.py
├── indicators.py
├── rules.py
├── signal_engine.py
├── charts.py
├── utils.py
├── requirements.txt
└── README.md
```

各文件说明如下：

### `app.py`

项目主入口文件，负责页面布局、交互逻辑、结果展示和图表渲染调用。

### `data_fetcher.py`

负责根据股票代码获取历史行情数据，并返回行情 DataFrame 与股票名称。

### `indicators.py`

负责基于原始行情数据计算各类技术指标，例如均线、MACD、RSI 和 5 日均量。

### `rules.py`

负责基于最新指标结果执行条件判断，输出规则判断结果。

### `signal_engine.py`

负责根据信号规则综合判断，生成最终分析结果。

### `charts.py`

负责构建收盘价图、均线图、成交量图、MACD 图等 Plotly 图表。

### `utils.py`

负责存放通用工具函数，例如股票代码校验、价格格式化等。

### `requirements.txt`

项目依赖文件，用于统一安装当前项目所需的 Python 包。

### `README.md`

项目说明文档。

## 四、安装步骤

### 1. 准备 Python 环境

建议使用 Python `3.9` 或更高版本。

可以先检查本机 Python 版本：

```bash
python --version
```

或者：

```bash
python3 --version
```

### 2. 进入项目目录

在终端中进入项目所在目录，例如：

```bash
cd /你的项目目录/量化工具
```

### 3. 安装项目依赖

执行以下命令安装依赖：

```bash
pip install -r requirements.txt
```

如果你的环境中默认使用 `python3`，也可以使用：

```bash
python3 -m pip install -r requirements.txt
```

建议在虚拟环境中安装，以避免与系统环境冲突。

## 五、启动方式

在项目根目录执行以下命令启动项目：

```bash
python -m streamlit run app.py
```

如果你的环境中需要使用 `python3`，则执行：

```bash
python3 -m streamlit run app.py
```

启动成功后，终端通常会输出本地访问地址，例如：

```text
http://localhost:8501
```

在浏览器中打开该地址即可使用项目。

## 六、使用方式

### 1. 打开页面

启动项目后，在浏览器中打开本地 Streamlit 页面。

### 2. 输入股票代码

在输入框中输入一个 6 位 A 股股票代码，例如：

- `600853`
- `000001`
- `002594`

### 3. 点击开始分析

点击“开始分析”按钮后，系统会自动完成以下流程：

- 获取历史行情数据
- 计算技术指标
- 执行规则判断
- 生成信号结果
- 展示图表和说明信息

### 4. 查看分析结果

页面分析完成后，可以查看：

- 当前股票名称和代码
- 当前收盘价格
- 当前信号结果
- 信号原因说明
- 满足条件列表
- 不满足条件列表
- 收盘价走势
- 均线走势
- 成交量变化
- MACD 动能图

## 七、运行说明与注意事项

### 1. 数据来源说明

本项目通过公开行情接口获取历史行情数据，用于个人学习和分析展示。

### 2. 使用范围说明

本项目仅用于本地个人分析，不用于自动交易、程序化下单或实盘执行。

### 3. 风险提示

本项目输出的信号仅为基于规则的技术分析结果，不构成任何投资建议。

### 4. 异常情况说明

如遇以下情况，系统可能提示错误信息：

- 股票代码格式错误
- 股票代码不存在
- 网络连接异常
- 数据接口暂时不可用
- 返回数据为空或字段缺失

出现上述情况时，可尝试：

- 检查股票代码是否正确
- 检查网络是否正常
- 稍后重新尝试
